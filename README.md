# Solutions in C for LeetCode
Previous code written in C++ for Zerojudge can be found in [shihlings/_archive-zerojudge](https://github.com/shihlings/_archive-zerojudge). They are now archived. This repository includes my solutions written in C to the [LeetCode questions](https://leetcode.com).

## Repository Management
### Branch Hierarchy
<b>All changes should be commited to 'staging', not 'main'.</b> Only verified and validated code should be pushed to 'main' via a pull request.

### Branch Protetion
'main' and 'staging' are protected and cannot be deleted.

### Commit Signing and Verification
All commits made to this repository are required to be signed. Any unsigned commits will be blocked.
